# Card

```tsx
import { Card } from '../index';
```

| Prop  | Type   | Default | Description                     |
| ----- | ------ | ------- | ------------------------------- |
| hover | bool   | true    | Enable hover tilt animation     |
| glass | bool   | false   | Glassmorphism style background  |

Usage:

```tsx
<Card>Content</Card>
```
