# Input

```tsx
import { Input } from '../index';
```

| Prop   | Type     | Description                     |
| ------ | -------- | -------------------------------- |
| label  | string   | Accessible label text            |
| error  | string   | Optional error message           |
| ...rest| HTMLInputProps | Native input props       |

Usage:

```tsx
<Input label="Email" name="email" />
```
