# Button

```tsx
import { Button } from '../index';
```

| Prop    | Type                         | Default   | Description    |
| ------- | ---------------------------- | --------- | -------------- |
| variant | "primary" | "secondary"        | "primary" | Visual style   |
| size    | "sm" | "md" | "lg"             | "md"      | Button size    |

Usage:

```tsx
<Button>Click me</Button>
```
